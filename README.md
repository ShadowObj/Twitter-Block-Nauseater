## 使用方式：
1. 在Greasyfork上安装脚本 (没有TamperMonkey等脚本管理器的按 [该向导](https://greasyfork.org/zh-CN) 安装)
2. 用电脑打开列表 -> [点击跳转到列表地址](https://twitter.com/i/lists/1685183911725252608)
3. 点击列表简介上的 **成员** (在关注者旁边)
4. 进入成员列表，**点击左上角 -> 屏蔽所有Nauseater**
